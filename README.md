Merci d'avoir téléchargé Piou Piou !
=================

Prérequis
----------

Pour jouer à Piou Piou, il faut que vous ayez téléchargé python 3.10 ou 
une version supérieure, ainsi que la bibliothèque pygame.

Les bibliothèques utilisées sont :
* Pygame
* Pickle
* Sys
* Os
* Logging
* Random
* Pathlib
* et Json
